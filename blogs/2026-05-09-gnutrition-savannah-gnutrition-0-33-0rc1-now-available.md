A test release of GNUtrition, 0.33.0rc1, is now available. GNUtrition is free nutrition analysis software written for the GNU operating system. The USDA Food and Nutrient Database for Dietary Studies (FNDDS) is used as the source of food nutrient information. This release is for the C rewrite, which is usable with GTK and ncurses based interfaces, along with a noninteractive mode.  The database was updated from the USDA DSR to the USDA FNDDS. Thank you, very much to Jason Self for providing us with the rewrite.  This release would not have been possible without it!
